# Electron Video Player

Electron desktop application that wraps a [video.js](https://videojs.com/) player for watching local media files.

## Prerequisites

- Node.js 18+
- npm 9+

## Getting Started

```bash
npm install
npm start
```

Use the **Open Videoâ€¦** button to browse for a file. Supported formats include MP4, MKV, MOV, WEBM, AVI, WMV, and more. Once selected, the video will begin playback in the embedded player. On macOS you can also open files by dragging them onto the app icon.
